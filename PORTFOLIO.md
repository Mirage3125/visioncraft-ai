# 面试作品介绍稿

## 项目名称
PixelForge AI：面向营销与电商场景的智能生图工作台

## 项目背景
普通 AI 生图演示通常只包含输入框和结果图片，缺少真实产品所需的参数配置、异常处理和作品管理。我将生图能力包装成一个完整工作台，用于帮助内容创作者快速产出商业摄影、营销海报和社交媒体视觉素材。

## 我的工作
- 独立完成品牌定位和 UI 设计
- 设计图片生成 API 与参数校验
- 接入 Vercel AI SDK 和 OpenAI 图像模型
- 实现无 API Key 的演示降级方案
- 实现本地历史记录、预览和下载
- 完成响应式适配与 Vercel 部署配置

## 技术难点
- API Key 必须只保存在服务端，避免前端泄漏
- 不同画幅需要映射到模型支持的实际尺寸
- 生图耗时较长，需要清晰的加载和失败反馈
- 面试现场网络或额度异常时，仍需保证产品流程可展示

## 后续规划
- Supabase Auth 与云端作品库
- 多模型并行对比
- 用户积分与调用成本控制
- 图片上传、图生图和局部重绘
- 内容安全审核和请求限流
